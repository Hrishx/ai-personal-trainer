# AI Personal Trainer

Real-time AI personal trainer using Computer Vision and Machine Learning for:

- Pose correction
- Rep counting
- Fatigue detection
- Injury risk analysis
- Real-time voice coaching

## Tech Stack
Python, OpenCV, MediaPipe, FastAPI, Streamlit, React/Next.js

## Features
- Live exercise tracking
- Posture correction feedback
- Rep phase detection
- Workout summary analytics
